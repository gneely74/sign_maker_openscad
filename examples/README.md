# Sign Maker Examples

This directory contains example configurations for different types of signs.

## Available Examples

### 1. snapon_style.scad
Classic industrial tool brand aesthetic with red base and white text.
- **Size**: Medium (40mm font)
- **Style**: Bold, geometric, all-caps
- **Holes**: Standard spacing (15mm)
- **Use case**: Tool branding, garage signs

### 2. garage_sign.scad
Large wall-mounted sign for garage or workshop.
- **Size**: Large (60mm font)
- **Style**: Bold sans-serif
- **Holes**: Wide spacing (40mm) with larger diameter
- **Use case**: Wall mounting, outdoor signage

### 3. nameplate.scad
Small desktop nameplate with elegant styling.
- **Size**: Small (20mm font)
- **Style**: Serif font, refined appearance
- **Holes**: Fine spacing (10mm) with small diameter
- **Use case**: Desk nameplates, door signs

### 4. no_holes_decorative.scad
Pure decorative sign without mounting holes.
- **Size**: Medium-large (50mm font)
- **Style**: Clean, modern
- **Holes**: None - use adhesive or stand
- **Use case**: Decorative pieces, freestanding displays

## Usage

1. Open any example file in OpenSCAD
2. Preview with F5 (fast) or Render with F6 (final quality)
3. Modify parameters at the top of the file to customize
4. Export as STL for 3D printing

## Customization Tips

- Change `text_string` to your desired text
- Adjust `font_size` for different scales
- Modify `hole_spacing` to match your mounting requirements
- Increase `$fn` for smoother circles (slower render)
- Try different fonts available on your system
